# Mapa maestro de productos químicos Chemie v1

## Objetivo
Crear una pieza maestra, estable y reutilizable para aprender el catálogo químico de Chemie con criterio, separando para cada producto:
- qué es
- qué función cumple
- dónde se aplica
- qué soporte documental tiene
- qué especificaciones técnicas visibles constan
- si tiene o no encaje eco documentado
- y qué dudas siguen abiertas

Este documento no pretende cerrar ya toda la clasificación técnica o legal definitiva.
Pretende dejar un **mapa prudente y gobernable** del núcleo fuerte inicial.

---

## Principio de trabajo
En este sector conviene distinguir siempre entre:
- **producto comercial**
- **familia funcional**
- **uso**
- **aplicación concreta**
- **dosificación o modo de uso** (solo si consta)
- **especificaciones técnicas**
- **encaje eco**
- **marco regulatorio prudente**

Si no se separan estas capas, se mezcla todo bajo “productos químicos” y se pierde completamente la lógica real del catálogo.

## Lecturas útiles que deja ya esta tanda

### 1. No todo el catálogo entra en el mismo carril
Con esta primera tanda ya se ve muy claro que mezclarlo todo como “productos químicos” sería un error.
Aquí conviven al menos:
- oxidantes sensibles
- desengrasantes industriales
- auxiliares de postlavado hortofrutícola
- aceites blancos alimentarios
- clorados de blanqueo/limpieza
- clorados de limpieza-desinfección más sensibles
- detergentes alcalinos de proceso

### 2. “Uso” y “aplicación” no son lo mismo
Ejemplos:
- **HIDROBRILL** puede tener uso de abrillantado/secado, pero aplicación en túneles o líneas hortofrutícolas
- **V-80** tiene uso de lubricación/auxilio técnico, pero aplicación en procesado y envasado alimentario
- **ALICLOR** tiene uso de limpieza-desinfección, pero aplicación sobre superficies/equipos/utensilios

### 3. “Eco” debe manejarse con mucho cuidado
En esta tanda, solo **OXISAN** tiene una conexión fuerte ya trabajada en el proyecto con el eje CAAE/ecológico.
En el resto, si no consta, lo correcto es **no inventarlo**.

### 4. La dosificación no debe rellenarse por intuición
Aunque probablemente muchos de estos productos tengan rangos o modos de uso en la documentación ampliada, en este mapa solo debe ponerse cuando conste de verdad.

## Conclusión ejecutiva
Esta primera tanda ya permite construir un aprendizaje serio del catálogo fuerte de Chemie.

La idea clave sería esta:

> **Para entender bien estos productos no basta con saber el nombre; hay que separar función, aplicación, soporte documental, sensibilidad regulatoria y, solo cuando conste, dosificación y encaje eco.**

Ese es el criterio bueno para que el mapa no se convierta en una lista plana de referencias.
