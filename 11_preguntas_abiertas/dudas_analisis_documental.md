# Dudas de análisis documental

## Propósito
Este archivo centraliza las dudas abiertas que van apareciendo durante la ingesta y análisis documental del proyecto `grupo_familiar`.

## Regla operativa
- En cada análisis o inspección documental, Minion debe registrar aquí las dudas nuevas relevantes.
- Si una duda se resuelve, el contenido confirmado debe pasarse al documento principal que corresponda:
  - ficha
  - inventario
  - mapa
  - memoria operativa
  - auditoría
  - u otro archivo base
- Una vez incorporada y consolidada la respuesta en el documento correcto, la duda debe eliminarse de este archivo o marcarse expresamente como resuelta si conviene dejar trazabilidad breve.
- No mezclar aquí conclusiones cerradas con hipótesis abiertas: este archivo es para incertidumbre, conflicto, huecos o validaciones pendientes.
