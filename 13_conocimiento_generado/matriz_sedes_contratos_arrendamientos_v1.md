# Matriz sedes - contratos - arrendamientos v1

## Objetivo
Ordenar las sedes y espacios reales de uso del núcleo (**Chemie, ACI y Ecoclean**) cruzando domicilio, actividad, propiedad y arrendamientos.

## Matriz

### Chemie -> Sierra de Almijara 5, Viator
- Tipo: **Nave/almacén principal**
- Estatus: **Propia o eje operativo consolidado**
- Base documental: Licencia/apertura, industria, PCI, sanidad, CAAE, financiación, actividad química
- Lectura: Núcleo espacial principal de Chemie

### ACI -> Sierra de Telar, La Juaida
- Tipo: **Domicilio/sede histórica**
- Estatus: **Histórico documental**
- Base documental: Cambio de domicilio detectado en 2000
- Lectura: Fase anterior al eje Almijara

### ACI -> Sierra de Almijara 3, Viator
- Tipo: **Referencia censal/documental secundaria**
- Estatus: **No prioritaria frente a la referencia consolidada**
- Base documental: Certificados censales IAE sitúan aquí actividad principal 617.9
- Lectura: Se mantiene como referencia documental histórica o censal, pero no desplaza la fijación operativa principal en Almijara 5

### ACI -> Sierra de Almijara 5, Viator
- Tipo: **Domicilio fiscal/social consolidado + nave**
- Estatus: **Propio / operativo-patrimonial consolidado**
- Base documental: Nave finca 5229, agua industrial, alarma, actividad empresarial y confirmación directa de Jero
- Lectura: Referencia buena y estable para el eje ACI/Chemie

### Ecoclean -> Avda. La Fabriquilla 3 bajo, El Parador
- Tipo: **Local comercial inicial**
- Estatus: **Arrendado**
- Base documental: Contrato de alquiler 2018, base de arranque comercial
- Lectura: Primera sede operativa clara

### Ecoclean -> Sierra de Loja 11, Viator
- Tipo: **Dirección comercial/fiscal posterior**
- Estatus: **Uso documental/operativo**
- Base documental: Aparece en documentación posterior
- Lectura: Conecta Ecoclean con eje Viator

### Ecoclean -> Sierra de Almijara 5, Viator
- Tipo: **Domicilio de entrega / eje logístico**
- Estatus: **Uso operativo indirecto**
- Base documental: Entrega de mercancía y relación con nave compartida
- Lectura: Más logístico que sede plena propia

### Ecoclean -> Nave compartida en Viator
- Tipo: **Espacio operativo intermedio**
- Estatus: **Arrendado ya cesado**
- Base documental: Anexo 2020: Ecoclean ocupa 25% aprox. 64 m², alquiler ya dejado según Jero
- Lectura: Fase intermedia ya cerrada

### Ecoclean -> Local actual pendiente
- Tipo: **Sede operativa actual**
- Estatus: **Pendiente de contrato**
- Base documental: Aclaración directa de Jero
- Lectura: Actualizar cuando llegue contrato

## Lecturas rápidas
- **Chemie** tiene el eje espacial más estable y consolidado: **Sierra de Almijara 5** como centro operativo claro.
- **ACI** comparte el mismo entorno espacial fuerte y debe fijarse operativamente en **Almijara 5**, quedando **Almijara 3** como referencia documental secundaria.
- **Ecoclean** es la pieza con cronología de sedes más móvil: **El Parador -> Viator -> local actual pendiente**.
- Viator no solo es un domicilio formal, sino el gran nodo espacial del sistema.

## Lectura espacial provisional
El grupo parece concentrar su densidad operativa en torno a **Viator / La Juaida / Sierra de Almijara**, mientras que El Parador funciona como base secundaria de arranque o despliegue para líneas más ligeras como Ecoclean.
