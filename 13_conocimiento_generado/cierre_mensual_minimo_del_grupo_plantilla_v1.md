# Cierre mensual mínimo del grupo - plantilla v1

## Objetivo
Servir como plantilla base para revisar el grupo cada mes de forma breve, útil y repetible.

No está pensada para hacer contabilidad formal ni informe bonito.
Está pensada para responder, mes a mes:
- qué ha pasado
- qué preocupa
- qué mejora
- y qué hay que vigilar o decidir
