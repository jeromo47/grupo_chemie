# Tabla maestra de mundos funcionales del catálogo químico Chemie v1

## Objetivo
Dejar una vista final, clara y muy consultable del sistema ya construido, cruzando:
- producto
- mundo funcional
- problema que resuelve
- dónde actúa
- lógica principal
- y principal riesgo de confusión

Esta tabla no sustituye a las fichas largas ni a las capas técnicas.
Las ordena en una vista estratégica de aprendizaje.

## Cómo leer esta tabla

### 1. Si quieres aprender el catálogo por mundos
- **Hortofrutícola/postcosecha** = OXISAN, TR-20, HIDROBRILL
- **Superficies/equipos/líneas** = CHEMGRAS, ALICLOR, ACTIV CLOR, TUNELIMP
- **Manos/sanitización** = DERGEL, LICOGEL
- **Auxiliares técnicos/lubricación** = V-80, IBERFARM L
- **Mantenimiento general** = BRINET
- **Solvente técnico** = NETIND

### 2. Si quieres aprender el catálogo por problema real
- **tratar proceso/agua** = OXISAN, TR-20
- **mejorar acabado** = HIDROBRILL
- **desengrasar** = CHEMGRAS DS
- **desengrasar + higienizar** = CHEMGRAS BAC
- **blanquear/limpiar con cloro** = ACTIV CLOR
- **limpiar/desinfectar con cloro** = ALICLOR
- **lavar línea/túnel** = TUNELIMP / TUNELIMP CL
- **lavar manos** = DERGEL SO
- **sanitizar rápido** = LICOGEL
- **lubricar/auxiliar proceso** = V-80 / IBERFARM L
- **mantenimiento general** = BRINET BQ / AML
- **disolver/limpieza solvente** = NETIND

### 3. Si quieres aprender el catálogo por errores de confusión
La última columna es muy útil para eso.
Porque gran parte del aprendizaje real consiste en **no mezclar referencias cercanas**.

---

## Lo más importante que deja esta tabla
### A. El catálogo tiene varios mundos funcionales de verdad
No parece una lista plana de productos equivalentes.

### B. La parte más fina sigue siendo la hortofrutícola/postcosecha
Porque ahí se cruzan:
- agua
- proceso
- residuo
- acabado
- posible eco

### C. La parte más amplia parece ser superficies/equipos/líneas
Y ahí conviene separar muy bien:
- desengrase
- higienización no clorada
- clorado simple
- clorado más sensible
- detergencia de línea

### D. El catálogo no es solo limpieza
Eso se ve clarísimo con:
- **V-80**
- **IBERFARM L**
- **OXISAN**
- **TR-20**
