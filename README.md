# Grupo familiar

## Objetivo
Construir una base documental y de conocimiento viva sobre Chemie, ACI y Ecoclean.

Esto no es solo un almacén de archivos.
Es un sistema para:
- subir documentos
- clasificarlos
- resumirlos
- detectar huecos
- construir contexto acumulado
- generar preguntas mejores
- saber qué falta por localizar

## Flujo recomendado
1. Todo documento nuevo entra por `00_inbox/`
2. Minion lo revisa y clasifica
3. Se mueve a la carpeta correcta
4. Se actualizan los archivos vivos de conocimiento
5. Se detecta qué sigue faltando

## Carpetas principales
- `00_inbox` -> entrada sin clasificar
- `01_chemie` -> docs de Chemie
- `02_aci` -> docs de ACI
- `03_ecoclean` -> docs de Ecoclean
- `04_personas_y_roles` -> organigrama, roles, personas clave
- `05_clientes` -> clientes, segmentación, dependencia
- `06_proveedores` -> proveedores, comparativas, dependencia
- `07_finanzas` -> ventas, márgenes, tesorería, cuentas, informes
- `08_fiscal_legal` -> fiscalidad, contratos, documentos legales
- `09_procesos` -> procesos operativos y mapas internos
- `10_reuniones` -> notas y actas
- `11_preguntas_abiertas` -> dudas que siguen vivas
- `12_documentos_faltantes` -> lista viva de lo que falta localizar
- `13_conocimiento_generado` -> resúmenes, mapas, síntesis, hallazgos
- `14_decisiones` -> decisiones relevantes del grupo
- `99_archivo` -> histórico
