# Revisión web Chemie: referencias clave v1

## Objetivo
Ver qué referencias clave del bloque químico:
- siguen visibles en la web actual de Chemie
- cómo se presentan hoy
- cuáles no aparecen o parecen haberse caído del escaparate online
- y qué productos conviene tener más cazados por combinación de negocio real + presencia o ausencia web

Metodología prudente:
- la web no sustituye a ventas ni a documentación técnica
- la ausencia en web no demuestra por sí sola que el producto no exista
- la presencia en web sí ayuda a detectar qué referencias siguen vivas comercialmente o cómo se están reencuadrando

---

## 1. Referencias claramente confirmadas en web

### OXISAN
Sí aparece en web.
La presentación online refuerza mucho la lectura ya construida:
- tratamiento de agua en industria alimentaria
- lavado postcosecha de frutas, verduras y hortalizas
- principio activo con peróxido de hidrógeno, ácido acético y ácido peracético
- lenguaje de **coadyuvante tecnológico**
- foco en contaminación cruzada, recirculación de agua y seguridad alimentaria
- además se presenta como apto para **producción ecológica**

**Lectura:**
La web no solo confirma OXISAN, sino que lo perfila incluso mejor.
Es una pieza central clarísima.

### ACTIV CLOR
Sí aparece en web.
La web lo presenta como:
- limpiador detergente clorado
- alto poder de limpieza
- apto para industria alimentaria
- **no contiene amonios cuaternarios**

**Lectura:**
Muy útil. La web refuerza que ACTIV CLOR sigue vivo y lo diferencia con un ángulo comercial interesante.

### TUNELIMP
Sí aparece en web.
Se mantiene como:
- detergente para túneles de lavado industrial
- industria alimentaria
- poder higienizante
- bajo espumante
- control de incrustaciones

**Lectura:**
Confirma que TUNELIMP está vivo comercialmente y con un discurso bastante alineado con la lectura documental.

### CHEMGRAS ASF
Aparece en web una variante **CHEMGRAS ASF**, no exactamente CHEMGRAS DS.
Se presenta como:
- desengrasante altamente concentrado
- bajo poder espumante
- uso industrial/hostelería/talleres
- apto para industria alimentaria

**Lectura:**
Esto es importante porque sugiere que la familia CHEMGRAS sí está muy viva online, pero puede haber cambio de referencias o énfasis comercial distinto respecto a la base de ventas histórica.

## 2. Referencias no encontradas claramente en la web actual
En la búsqueda web directa no han aparecido de forma clara estas referencias:
- **CHEMGRAS BAC**
- **BACTER CHEM**
- **IBERFARM L**
- **LICOGEL**
- **BRINET L**
- **TR-20**
- **DERGEL SO**
- **V-80**
- **TUNELIMP CL**

## Lectura prudente
Esto no significa automáticamente que estén muertas.
Puede significar:
- producto activo pero no destacado en la web pública
- producto antiguo o de nicho
- producto aún comercializable pero desplazado por otra denominación
- referencia histórica de catálogo no priorizada online
- producto vivo en ventas B2B pero poco expuesto en la web

## 3. Qué ausencias web me parecen más interesantes
### 1. V-80
Es probablemente la ausencia más llamativa.
Porque en ventas sí sale fuerte y en la documentación técnica también merece atención.

**Lectura:**
V-80 debería quedar muy cazado aunque no aparezca online, precisamente porque combina:
- peso comercial real
- especificación técnica seria
- ausencia de escaparate web

Eso puede indicar producto muy B2B o muy técnico, poco pensado para marketing abierto.

### 2. DERGEL SO
También llama la atención que no aparezca claramente, porque en ventas tiene mucha recurrencia.

**Lectura:**
Puede ser un producto muy operativo de rotación real, pero poco visible en la fachada web.

### 3. TR-20
No sale online de forma clara, pero documentalmente es muy específico.

**Lectura:**
Aunque no pesa mucho en ventas visibles, su valor explicativo del bloque hortofrutícola sigue siendo alto.

### 4. IBERFARM L
No aparece online y tampoco salió fuerte por nombre en el cruce simple de ventas.

**Lectura:**
Aquí baja un poco la prioridad económica inmediata, pero no desaparece el interés técnico.
Puede ser más de soporte, histórico o nicho.

### 5. CHEMGRAS BAC y BACTER CHEM
No salen bien en la web y tampoco salieron fuertes por nombre en el cruce básico.

**Lectura:**
Mantienen interés técnico, pero pierden urgencia comercial frente a otras referencias ya confirmadas por ventas + web.

### 6. TUNELIMP CL
No aparece como tal, mientras que **TUNELIMP** sí aparece online.

**Lectura:**
Esto refuerza la idea de que TUNELIMP CL puede ser:
- variante secundaria
- referencia específica menos visible
- o formulación comercial no priorizada en la web

## 6. Juicio final
La web sirve más para **confirmar focos** que para descubrir grandes productos ocultos.

La conclusión fuerte no cambia demasiado, pero sí se afina:
- **OXISAN**, **ACTIV CLOR** y **TUNELIMP** están claramente vivos online
- la familia **CHEMGRAS** sigue viva, aunque quizá con referencias comerciales algo mutadas
- productos como **V-80** o **DERGEL SO** ganan interés precisamente porque **sí pesan en ventas aunque no brillen en web**
- y varias referencias de catálogo interesante quedan ahora en una zona de **interés técnico, pero urgencia comercial menor**

## Cierre corto
Si tuviera que resumir esta capa web en una frase:

> La web confirma bien el núcleo fuerte ya conocido, pero los productos que de verdad conviene vigilar no son solo los más visibles online, sino los que combinan documentación sólida, peso comercial real y sentido técnico dentro del mapa químico de Chemie.
