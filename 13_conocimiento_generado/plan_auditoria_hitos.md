# Plan de auditoría por hitos

## Enfoque
La auditoría no se hará por calendario rígido, sino por hitos de conocimiento. Esto reduce burocracia y mejora calidad real.

## Regla de seguimiento continuo
Además de las auditorías por hitos, Minion debe ir dejando por escrito:
- **hitos relevantes ya consolidados**
- **cambios de interpretación importantes**
- **nuevos frentes de auditoría sugeridos**
- y **pendientes de mejor extracción desde base de datos o fuente más fina**

La idea es que el proyecto no dependa de memoria conversacional, sino de rastro documental acumulado.

## Hitos propuestos

### Hito 1. Consolidación de Chemie formal
Incluye:
- Escritura de constitución
- Ampliación de capital
- Poder mercantil
- Vinculación jurídica
- Primera ficha formal de Chemie

Auditar:
- calidad de extracción
- trazabilidad de datos societarios
- separación entre hecho e inferencia
- huecos documentales

### Hito 2. Licencias y registros de Chemie
Incluye:
- Licencia apertura GCH
- Registro biocidas
- Registro sanitario
- Registro ecológico
- Sanidad

Auditar:
- vigencia
- entidad titular
- dirección/actividad
- coherencia entre registros

### Hito 3. Bloque patrimonial prioritario
Incluye revisión de activos de mayor valor o mayor ambigüedad.

Auditar:
- evidencia mínima por activo
- estado actual
- entidad o persona asociada
- si la asociación es documental o inferida

### Hito 4. ACI
Auditar estructura, fiabilidad documental y vacíos.

### Hito 5. Ecoclean
Auditar estructura, fiabilidad documental y vacíos.

### Hito 6. Auditoría global pre-septiembre
Objetivo:
- saber qué está sólido
- qué queda abierto
- qué requiere decisión tuya
- qué faltaría para usar todo esto como base de gestión real
