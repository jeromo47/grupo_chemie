- qué es
- para qué sirve
- cómo se usa

- si entra o no en eco
- y en qué se diferencia de otros del mismo sector o familia

## Qué conviene diferenciar en este sector
Aquí está la clave.
Si no se separan bien estas capas, se mezcla todo y se vuelve imposible aprender el catálogo con criterio.

### 1. Producto comercial
Es el nombre con el que se vende:
- OXISAN
- CHEMGRAS DS
- HIDROBRILL
- ALICLOR COMPLET
- etc.

### 2. Familia funcional
Qué hace, de forma prudente:
- desengrasante
- detergente técnico
- clorado limpiador/blanqueante
- oxidante
- abrillantador/postlavado
- lubricante técnico alimentario
- higienizante manos
- disolvente
- etc.

### 3. Uso o aplicación real
Dónde o para qué se usa:
- lavado de frutas y hortalizas
- aguas de lavado
- túneles de lavado
- superficies y equipos
- industria alimentaria
- limpieza general
- mantenimiento
- manos
- lubricación/auxiliar de proceso

### 4. Forma de intervención
Esto también hay que separarlo bien:
- producto para **limpiar**
- producto para **desengrasar**
- producto para **blanquear/clorar**
- producto para **desinfectar**
- producto para **higienizar**
- producto para **abrillantar/posttratar**
- producto para **lubricar o ayudar al proceso**
- producto para **tratar aguas**

### 5. Soporte documental disponible
No vale lo mismo:
- **FDS real**
- ficha técnica/comercial
- CLP resumido
- referencia web
- naming en ventas

### 6. Dosificación / modo de uso
Aquí hay que ser muy prudentes.
Solo debe recogerse cuando **conste en la documentación**.
No se debe inventar ni extrapolar por costumbre del sector.

### 7. Especificaciones o claims técnicos
Por ejemplo:
- apto industria alimentaria
- uso profesional
- cloro activo
- alcalino
- bioalcohol
- amoniacal
- aceite blanco medicinal
- compatibilidad con ciertas superficies o entornos

### 8. Encaje eco / ecológico
Aquí también hay que separar varias cosas distintas:
- producto expresamente conectado con **CAAE / eco**
- producto útil en entorno alimentario pero **sin soporte eco**
- producto sin indicio eco
- producto dudoso

Muy importante:
**“se usa en agro/alimentario” no equivale automáticamente a “vale para eco”.**

## Cómo distinguir bien “producto”, “uso”, “aplicación” y “eco”
### Producto
Qué es la referencia comercial concreta.

### Uso
Qué función cumple.
Ejemplo:
- desengrasar
- higienizar
- abrillantar
- tratar agua

### Aplicación
Dónde o en qué proceso se usa.
Ejemplo:
- túnel de lavado
- agua de proceso
- manos
- superficies
- hortofrutícola

### Dosificación
Cuánto o en qué rango se usa, **solo si el documento lo dice**.

### Especificación
Qué rasgo técnico relevante lo define.
Ejemplo:
- alcalino
- clorado
- oxidante
- aceite blanco medicinal
- uso profesional

### Eco
Si existe soporte documental real para:
- compatible con producción ecológica
- coadyuvante tecnológico con apoyo eco
- o si **no consta**
